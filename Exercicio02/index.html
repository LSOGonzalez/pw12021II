Guia de aprendizado em Programação Web

Cabeçalho

  • A tag <!DOCTYPE> representa o tipo de documento
  • A tag <html> é a raíz de um documento html, é container de todos os elementos (menos !DOCTYPE) e
    dentro dela deve sempre ser declarado um idioma
  • A tag <head> contém metadata/informação para o documento
  • A tag <meta> define a metadata/informação para o documento
  • A tag <link> estabelece uma ligação entre o documento e uma fonte externa. Ex: folha de estilo CSS
  • A tag <title> define o título do documento, aparece na aba da página

O Corpo

  • Dentro da tag body encontra-se todo o conteúdo de um documento HTML, como heading, parágrafos
    imagens, hyperlinks, tabelas, listas e etc.
  • as tags de h1 até h6, são usadas para definir as headings do documento. Sendo h1 a maior e "mais importante"
    e h6 a menor e "menos importante"
  • a tag <p> define um parágrafo. O navegador automaticamente adiciona uma blank line depois de cada <p>
  • a tag <span> é usada para colorir uma parte de um texto, é um inline container e é facilmente manipulada por CSS
  • tag <img> é a tag usada para adicionar uma imagem ao documento html, onde possui dois requisitos:
    1° scr = "" especifica a origem do documento
    2° alt = "" descreve a imagem caso ela não carregue
